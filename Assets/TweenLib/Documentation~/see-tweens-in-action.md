# See Tweens in action

1. Clone the project.
2. Open the [SampleScene](../../Scenes/SampleScene.unity).
3. Enter the PlayMode
4. Modify tweening configs as you wish and press `Space` to trigger normal Tween, press `F` to trigger Shake Tween
